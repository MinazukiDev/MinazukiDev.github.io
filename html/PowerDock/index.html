PowerDock

Description

  • Add a power button below the dock on the home screen. This allows you to quickly respring.

    Hide Dock Blur
    Hide the dock blur.

    Corner Radius
    Change the corner blur of the PowerDock button.

    Dock Height
    Change the height of the dock.

    Font Size
    Changes the size of the text in the PowerDock button.

    Title Alphe
    Changes the alpha value of the PowerDock button's text.

Compatibility

  • iOS 13.0 - 13.4
